# study-mysql

Test environment MySQL (5.7 or 8.0)

## Up MySQL5.7
```shell script
$ docker-compose -f docker-compose-v57.yaml up
```

## Up MySQL8.0
```shell script
$ docker-compose -f docker-compose-v80.yaml up
```

## MySQL Environment
- MYSQL_USER: root
- MYSQL_ROOT_PASSWORD: mysql
- MYSQL_PASSWRD: mysql
